# Task - Task and Project Manager

##  How to Run?

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/TaskNico.git
   cd TaskNico
   ```
2. **Install dependencies**
   ```bash
   pip install pandas smtplib
   ```
3. **Run the program**
   ```bash
   python main.py
   ```
